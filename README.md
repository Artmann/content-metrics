# Content Metrics

Displays revenue, impressions and visits

![](https://i.imgur.com/Vx68cyl.png)

## How do I run the app?

Install dependecies

```
yarn
```

Start the dev server

```
yarn start
```

Visit `http://localhost:8081/`

## How do I build the app for production

Install dependecies

```
yarn
```

Run tests

```
yarn test
```

Build the app

```
yarn build
```
